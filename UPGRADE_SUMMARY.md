## 🚀 Vercel 部署

### 部署步骤（5 分钟）

1. **推送代码到 GitHub**
   ```bash
   git add .
   git commit -m "feat: v2.0 优化 + Vercel 部署"
   git push
   ```

2. **Vercel 导入项目**
   - 访问 https://vercel.com
   - Import Git Repository
   - 选择 `problem-tree-tool`

3. **配置环境变量**
   ```
   DEEPSEEK_API_KEY=sk-xxx
   ANTHROPIC_API_KEY=sk-ant-xxx (可选)
   ```

4. **部署完成**
   - 获得 URL: `https://problem-tree-tool-xxx.vercel.app`
   - 访问测试

## 🎯 用户旅程（v2.0）

```
1. 访问首页 (0 秒)
   ↓
2. 输入问题 / 选择示例 (30 秒)
   ↓
3. 点击"开始拆解" (1 秒)
   ↓
4. 看到第 1 层洞察 (30 秒) ← **第一个 Aha 时刻**
   ↓
5. 继续探索或跳过 (可控)
   ↓
6. 完成 7 步或中途导出
   ↓
7. 下载报告/分享
```

**关键 Aha 时刻**:
- 第 4 步：30 秒看到第一个洞察
- 第 5 步：发现从未想到的视角
- 第 7 步：完整问题树的震撼
